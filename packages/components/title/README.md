# @loom/title

Loom Design System - Title Component

## Installation

```bash
npm install @loom/title
# or
yarn add @loom/title
# or
pnpm add @loom/title
```

## Usage

```tsx
import { Title } from '@loom/title';

function App() {
  return (
    <Title>
      Hello World
    </Title>
  );
}
```
