# @loom/avatar

Loom Design System - Avatar Component

## Installation

```bash
npm install @loom/avatar
# or
yarn add @loom/avatar
# or
pnpm add @loom/avatar
```

## Usage

```tsx
import { Avatar } from '@loom/avatar';

function App() {
  return (
    <Avatar>
      Hello World
    </Avatar>
  );
}
```
