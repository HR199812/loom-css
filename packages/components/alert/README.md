# @loom/alert

Loom Design System - Alert Component

## Installation

```bash
npm install @loom/alert
# or
yarn add @loom/alert
# or
pnpm add @loom/alert
```

## Usage

```tsx
import { Alert } from '@loom/alert';

function App() {
  return (
    <Alert>
      Hello World
    </Alert>
  );
}
```
