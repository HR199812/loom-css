# @loom/calendar

Loom Design System - Calendar Component

## Installation

```bash
npm install @loom/calendar
# or
yarn add @loom/calendar
# or
pnpm add @loom/calendar
```

## Usage

```tsx
import { Calendar } from '@loom/calendar';

function App() {
  return (
    <Calendar>
      Hello World
    </Calendar>
  );
}
```
