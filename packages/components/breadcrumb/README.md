# @loom/breadcrumb

Loom Design System - Breadcrumb Component

## Installation

```bash
npm install @loom/breadcrumb
# or
yarn add @loom/breadcrumb
# or
pnpm add @loom/breadcrumb
```

## Usage

```tsx
import { Breadcrumb } from '@loom/breadcrumb';

function App() {
  return (
    <Breadcrumb>
      Hello World
    </Breadcrumb>
  );
}
```
