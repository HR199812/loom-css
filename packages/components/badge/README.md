# @loom/badge

Loom Design System - Badge Component

## Installation

```bash
npm install @loom/badge
# or
yarn add @loom/badge
# or
pnpm add @loom/badge
```

## Usage

```tsx
import { Badge } from '@loom/badge';

function App() {
  return (
    <Badge>
      Hello World
    </Badge>
  );
}
```
