# @loom/radio

Loom Design System - Radio Component

## Installation

```bash
npm install @loom/radio
# or
yarn add @loom/radio
# or
pnpm add @loom/radio
```

## Usage

```tsx
import { Radio } from '@loom/radio';

function App() {
  return (
    <Radio>
      Hello World
    </Radio>
  );
}
```
