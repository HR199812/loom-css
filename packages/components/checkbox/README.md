# @loom/checkbox

Loom Design System - Checkbox Component

## Installation

```bash
npm install @loom/checkbox
# or
yarn add @loom/checkbox
# or
pnpm add @loom/checkbox
```

## Usage

```tsx
import { Checkbox } from '@loom/checkbox';

function App() {
  return (
    <Checkbox>
      Hello World
    </Checkbox>
  );
}
```
