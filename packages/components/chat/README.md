# @loom/chat

Loom Design System - Chat Component

## Installation

```bash
npm install @loom/chat
# or
yarn add @loom/chat
# or
pnpm add @loom/chat
```

## Usage

```tsx
import { Chat } from '@loom/chat';

function App() {
  return (
    <Chat>
      Hello World
    </Chat>
  );
}
```
