# @loom/input

Loom Design System - Input Component

## Installation

```bash
npm install @loom/input
# or
yarn add @loom/input
# or
pnpm add @loom/input
```

## Usage

```tsx
import { Input } from '@loom/input';

function App() {
  return (
    <Input>
      Hello World
    </Input>
  );
}
```
