# @loom/commandpalette

Loom Design System - CommandPalette Component

## Installation

```bash
npm install @loom/commandpalette
# or
yarn add @loom/commandpalette
# or
pnpm add @loom/commandpalette
```

## Usage

```tsx
import { CommandPalette } from '@loom/commandpalette';

function App() {
  return (
    <CommandPalette>
      Hello World
    </CommandPalette>
  );
}
```
