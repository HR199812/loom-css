# @loom/toast

Loom Design System - Toast Component

## Installation

```bash
npm install @loom/toast
# or
yarn add @loom/toast
# or
pnpm add @loom/toast
```

## Usage

```tsx
import { Toast } from '@loom/toast';

function App() {
  return (
    <Toast>
      Hello World
    </Toast>
  );
}
```
