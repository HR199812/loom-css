# @loom/lists

Loom Design System - Lists Component

## Installation

```bash
npm install @loom/lists
# or
yarn add @loom/lists
# or
pnpm add @loom/lists
```

## Usage

```tsx
import { Lists } from '@loom/lists';

function App() {
  return (
    <Lists>
      Hello World
    </Lists>
  );
}
```
