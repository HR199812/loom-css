# @loom/otpinput

Loom Design System - OTPInput Component

## Installation

```bash
npm install @loom/otpinput
# or
yarn add @loom/otpinput
# or
pnpm add @loom/otpinput
```

## Usage

```tsx
import { OTPInput } from '@loom/otpinput';

function App() {
  return (
    <OTPInput>
      Hello World
    </OTPInput>
  );
}
```
