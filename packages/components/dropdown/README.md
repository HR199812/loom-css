# @loom/dropdown

Loom Design System - Dropdown Component

## Installation

```bash
npm install @loom/dropdown
# or
yarn add @loom/dropdown
# or
pnpm add @loom/dropdown
```

## Usage

```tsx
import { Dropdown } from '@loom/dropdown';

function App() {
  return (
    <Dropdown>
      Hello World
    </Dropdown>
  );
}
```
