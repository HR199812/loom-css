# @loom/table

Loom Design System - Table Component

## Installation

```bash
npm install @loom/table
# or
yarn add @loom/table
# or
pnpm add @loom/table
```

## Usage

```tsx
import { Table } from '@loom/table';

function App() {
  return (
    <Table>
      Hello World
    </Table>
  );
}
```
