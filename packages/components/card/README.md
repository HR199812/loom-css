# @loom/card

Loom Design System - Card Component

## Installation

```bash
npm install @loom/card
# or
yarn add @loom/card
# or
pnpm add @loom/card
```

## Usage

```tsx
import { Card } from '@loom/card';

function App() {
  return (
    <Card>
      Hello World
    </Card>
  );
}
```
