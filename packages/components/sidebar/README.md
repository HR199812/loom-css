# @loom/sidebar

Loom Design System - Sidebar Component

## Installation

```bash
npm install @loom/sidebar
# or
yarn add @loom/sidebar
# or
pnpm add @loom/sidebar
```

## Usage

```tsx
import { Sidebar } from '@loom/sidebar';

function App() {
  return (
    <Sidebar>
      Hello World
    </Sidebar>
  );
}
```
