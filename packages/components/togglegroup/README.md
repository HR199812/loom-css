# @loom/togglegroup

Loom Design System - ToggleGroup Component

## Installation

```bash
npm install @loom/togglegroup
# or
yarn add @loom/togglegroup
# or
pnpm add @loom/togglegroup
```

## Usage

```tsx
import { ToggleGroup } from '@loom/togglegroup';

function App() {
  return (
    <ToggleGroup>
      Hello World
    </ToggleGroup>
  );
}
```
