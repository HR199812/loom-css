# @loom/accordion

Loom Design System - Accordion Component

## Installation

```bash
npm install @loom/accordion
# or
yarn add @loom/accordion
# or
pnpm add @loom/accordion
```

## Usage

```tsx
import { Accordion } from '@loom/accordion';

function App() {
  return (
    <Accordion>
      Hello World
    </Accordion>
  );
}
```
