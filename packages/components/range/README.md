# @loom/range

Loom Design System - Range Component

## Installation

```bash
npm install @loom/range
# or
yarn add @loom/range
# or
pnpm add @loom/range
```

## Usage

```tsx
import { Range } from '@loom/range';

function App() {
  return (
    <Range>
      Hello World
    </Range>
  );
}
```
