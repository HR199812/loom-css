# @loom/switch

Loom Design System - Switch Component

## Installation

```bash
npm install @loom/switch
# or
yarn add @loom/switch
# or
pnpm add @loom/switch
```

## Usage

```tsx
import { Switch } from '@loom/switch';

function App() {
  return (
    <Switch>
      Hello World
    </Switch>
  );
}
```
