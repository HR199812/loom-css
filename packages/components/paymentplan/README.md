# @loom/paymentplan

Loom Design System - PaymentPlan Component

## Installation

```bash
npm install @loom/paymentplan
# or
yarn add @loom/paymentplan
# or
pnpm add @loom/paymentplan
```

## Usage

```tsx
import { PaymentPlan } from '@loom/paymentplan';

function App() {
  return (
    <PaymentPlan>
      Hello World
    </PaymentPlan>
  );
}
```
