# @loom/separator

Loom Design System - Separator Component

## Installation

```bash
npm install @loom/separator
# or
yarn add @loom/separator
# or
pnpm add @loom/separator
```

## Usage

```tsx
import { Separator } from '@loom/separator';

function App() {
  return (
    <Separator>
      Hello World
    </Separator>
  );
}
```
