# @loom/grid

Loom Design System - Grid Component

## Installation

```bash
npm install @loom/grid
# or
yarn add @loom/grid
# or
pnpm add @loom/grid
```

## Usage

```tsx
import { Grid } from '@loom/grid';

function App() {
  return (
    <Grid>
      Hello World
    </Grid>
  );
}
```
