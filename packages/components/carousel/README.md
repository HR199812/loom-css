# @loom/carousel

Loom Design System - Carousel Component

## Installation

```bash
npm install @loom/carousel
# or
yarn add @loom/carousel
# or
pnpm add @loom/carousel
```

## Usage

```tsx
import { Carousel } from '@loom/carousel';

function App() {
  return (
    <Carousel>
      Hello World
    </Carousel>
  );
}
```
