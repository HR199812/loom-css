# @loom/searchinput

Loom Design System - SearchInput Component

## Installation

```bash
npm install @loom/searchinput
# or
yarn add @loom/searchinput
# or
pnpm add @loom/searchinput
```

## Usage

```tsx
import { SearchInput } from '@loom/searchinput';

function App() {
  return (
    <SearchInput>
      Hello World
    </SearchInput>
  );
}
```
