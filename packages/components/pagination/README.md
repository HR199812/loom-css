# @loom/pagination

Loom Design System - Pagination Component

## Installation

```bash
npm install @loom/pagination
# or
yarn add @loom/pagination
# or
pnpm add @loom/pagination
```

## Usage

```tsx
import { Pagination } from '@loom/pagination';

function App() {
  return (
    <Pagination>
      Hello World
    </Pagination>
  );
}
```
