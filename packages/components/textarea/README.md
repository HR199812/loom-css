# @loom/textarea

Loom Design System - TextArea Component

## Installation

```bash
npm install @loom/textarea
# or
yarn add @loom/textarea
# or
pnpm add @loom/textarea
```

## Usage

```tsx
import { TextArea } from '@loom/textarea';

function App() {
  return (
    <TextArea>
      Hello World
    </TextArea>
  );
}
```
