# @loom/modal

Loom Design System - Modal Component

## Installation

```bash
npm install @loom/modal
# or
yarn add @loom/modal
# or
pnpm add @loom/modal
```

## Usage

```tsx
import { Modal } from '@loom/modal';

function App() {
  return (
    <Modal>
      Hello World
    </Modal>
  );
}
```
