# @loom/section

Loom Design System - Section Component

## Installation

```bash
npm install @loom/section
# or
yarn add @loom/section
# or
pnpm add @loom/section
```

## Usage

```tsx
import { Section } from '@loom/section';

function App() {
  return (
    <Section>
      Hello World
    </Section>
  );
}
```
